#Calculadora en Python

Este es mi primer proyecto subido a GitHub como práctica de Python.

Se trata de una calculadora básica que permite realizar las siguientes operaciones:

- Suma  
- Resta  
- Multiplicación  
- División  

#Funcionalidades

- Uso de `match` 
- Validación de operador incorrecto
- Control de división entre 0
- Entrada de datos por consola
